# AI Leave Letter Generator

The AI Leave Letter Generator is a Generative AI application that automatically creates a formal leave letter based on user input.
The application uses Google Gemini API for text generation and Streamlit to provide a simple web interface where users can enter their details.

This project demonstrates how Generative AI models can be used to generate structured academic communication automatically.


## Features

Interactive Streamlit web interface

Automatically generates formal leave letters

Uses Google Gemini API for AI-powered text generation

Clean and properly formatted output

Secure API key handling using Streamlit secrets


## Technologies
- Python
- Streamlit
- Google Gemini API

  
## Learning Outcomes

Understanding Generative AI workflow

Using prompt engineering to control output formatting

Building a simple AI-powered web application

Integrating Gemini API with Streamlit
